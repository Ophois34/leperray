Commandes GIT

  • GIT STATUS
    : voir les fichiers en attente d'ajout
  • GIT ADD fichier
    : ajouter un fichier à l'index GIT
  • GIT ADD .
    : ajouter tous les fichiers modifiés à l'index GIT
  • GIT COMMIT -m "commentaire"
    : faire un commit
  • GIT DIFF
    : voir les différences entre l'index et le fichier
